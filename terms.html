🏠 Dom 🍽️ Meni Inženjer 👨‍🍳 O meni

Poslednja izmena: Maj 2026.

1. Prihvatanje uslova

Korišćenjem Meni Inženjer API-ja prihvatate ove uslove korišćenja. Ako se ne slažete sa uslovima, nemojte koristiti API.

2. Opis usluge

Meni Inženjer API je alat koji generiše opise jela i pića na osnovu unetih podataka (naziv jela, vrsta restorana, opcioni sastojci). API je dostupan kroz RapidAPI platformu.

3. Planovi i naplata

  • BASIC (besplatan): 50 opisa mesečno.
  • PRO (29€/mesečno): 500 opisa mesečno.
  • ULTRA (79€/mesečno): 2.000 opisa mesečno.

Sve cene su izražene u evrima (€). Naplata se vrši preko RapidAPI platforme. Promena plana stupa na snagu odmah.

4. Dozvoljeno korišćenje

API smete da koristite za:

  • Generisanje opisa jela za sopstvene menije
  • Integraciju u aplikacije za restorane i dostavu hrane
  • Komercijalne i nekomercijalne projekte

Nije dozvoljeno:

  • Preprodaja API-ja ili njegovih izlaza kao zasebnog proizvoda
  • Zloupotreba (spam, prekomerni zahtevi)
  • Korišćenje za generisanje neprikladnog sadržaja

5. Ograničenje odgovornosti

Meni Inženjer API se pruža "kako jeste". Ne garantujemo da će opisi uvek biti savršeni ili prikladni za svaku namenu. Korisnik je odgovoran za pregled i odobravanje generisanih opisa pre objavljivanja.

6. Zaštita podataka

API ne skladišti unete podatke (nazive jela, sastojke). Svaki zahtev se obrađuje u realnom vremenu i ne čuva se.

7. Izmene uslova

Zadržavamo pravo da izmenimo ove uslove. O značajnim izmenama obavestićemo korisnike putem RapidAPI platforme.
